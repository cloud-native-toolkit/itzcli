## itz solution list

Lists your IBM Technology Zone solutions.

### Synopsis

Lists the solutions for your IBM Technology Zone user.

```
itz solution list [flags]
```

### Options

```
  -h, --help       help for list
  -a, --list-all   If true, lists all the solutions available.
```
